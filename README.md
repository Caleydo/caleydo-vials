=====================

Vials visualizes alternative splicing based on mRNAseq data. You can try out vials at http://vials.io/.

Installation
------------

```
git clone https://github.com/caleydo/vials.git
cd vials
npm install
```

Testing
-------

```
npm test
```

Building
--------

```
npm run build
```
